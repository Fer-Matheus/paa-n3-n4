# Unbounded Knapsack

Implementação dos algoritmos do problema de otimização unbounded knapsack, onde a pool de elementos possiveis de escolha continua a mesma.

## Algoritmos implementados:
### Recursiva simples 
    - Complexidade O(n^(capacidade/min(pesos))
### Recursiva com memorização
    - Complexidade O(n * capacidade)
### Programação dinâmica
    - Complexidade O(n * capacidade)
    
## Aplicações
- Alocação de recursos com disponibilidade ilimitada
- Planejamento de produção industrial
- Corte de materiais (barras, cabos, chapas)
- Otimização de orçamento
- Logística e transporte com itens repetíveis
- Seleção de componentes em sistemas embarcados
- Mistura de produtos e formulações químicas
- Problemas de empacotamento e armazenamento
- Planejamento de investimentos com múltiplas unidades do mesmo ativo
