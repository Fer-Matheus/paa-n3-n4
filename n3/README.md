# Unbounded Knapsack

Implementação dos algoritmos do problema de otimização unbounded knapsack, onde a pool de elementos possiveis de escolha continua a mesma.

## Algoritmos implementados:
### Recursiva simples 
    - Complexidade O(n^(capacidade/min(pesos))
### Recursiva com memorização
    - Complexidade O(n * capacidade)
### Programação dinâmica
    - Complexidade O(n * capacidade)
